Submit a course

Create your course in Google Earth and submit it here.

  1. In Google Earth, create a Folder with your course name.
  2. Add Placemarks for start, waypoints, and finish. Draw a polygon for each (right-click Placemark → Add → Polygon).
  3. Export the Folder as KML (right-click → Save Place As).
  4. Upload the KML file below.
