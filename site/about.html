About

rownative.icu is a community-maintained library of GPS-defined rowing courses. Each course is a set of polygons (start, waypoints, finish) that define a measured course on the water.

What it's for

  • CrewNerd — The CrewNerd app uses these courses for automatic course timing and navigation. Sign in with your intervals.icu account to like courses and sync them to CrewNerd.
  • intervals.icu — We use intervals.icu for sign-in only. Your workout data stays in intervals.icu.

Open source

This project is open source. The course data and site code live in public repositories:

  • github.com/rownative/courses — Course data (JSON, KML), site, validation scripts, and GitHub Actions
  • github.com/rownative/worker — Cloudflare Worker (API, OAuth, submit/import)

Contributing

You can add courses via the Submit form or by opening a pull request. See How to contribute in the repository.
