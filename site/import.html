Import from Rowsandall

If you have courses on Rowsandall, you can migrate them here.

  1. On Rowsandall, go to your courses and use Download my courses to get a ZIP file.
  2. Upload that ZIP file below.

We will restore your liked courses and open pull requests for owned courses that are not yet in the library.
