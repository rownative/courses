Update a course

Submit a revised KML to update an existing provisional course. Only provisional courses can be updated via this form.

  1. In Google Earth, edit your course (or create a new version) and export the Folder as KML.
  2. Enter the course ID (from the course detail panel on the map) and upload the KML file below.
